# Rule-Based AI Chatbot 🤖
**Project 1 | DecodeLabs Internship | Batch 2026**
**Author: Soujoud Gahgah**

## About
A simple rule-based chatbot built with Python that responds to user input using keyword matching and a dictionary-based knowledge base. No machine learning — pure logic and control flow.

## How To Run
```bash
python rule_based_chatbot.py
```

## Features
- Time-aware greeting (morning / afternoon / evening)
- 17 intents with random response variation
- Input sanitization
- Fallback response for unknown inputs
- Conversation counter on exit

## Example
```
Bot: Good evening! 🌙 I'm RuleBot. Type 'help' for topics, or 'bye' to leave.
You: hello
Bot: Hi there! Ready to talk AI today?
You: joke
Bot: Why do programmers prefer dark mode? Because light attracts bugs!
You: bye
Bot: Goodbye! We exchanged 2 messages today. See you next time! 👋
```
